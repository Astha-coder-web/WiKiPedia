📚 Wikipedia Search Application

A simple and responsive Wikipedia Search Web App built using HTML, CSS, and JavaScript.
This application fetches real-time search results from the Wikipedia API using a GET request and dynamically displays article titles with links to their Wikipedia pages.

🚀 Features

🔎 Search any topic from Wikipedia

🌐 Fetches real-time data using Wikipedia API

⚡ Uses JavaScript Fetch API (GET request)

📱 Responsive and clean UI

🔗 Clickable links to open full Wikipedia articles

⏳ Loading spinner while fetching results

🛠️ Technologies Used

HTML5 – Structure of the application

CSS3 – Styling and layout

JavaScript (ES6) – Functionality and API calls

Wikipedia REST API – Fetching search results

⚙️ How It Works

User enters a keyword in the search bar.

JavaScript sends a GET request to the Wikipedia API.

API returns related search results.

Results are dynamically displayed on the page.

Clicking a result opens the Wikipedia article.
